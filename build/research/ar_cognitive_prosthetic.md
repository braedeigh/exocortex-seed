# AR Cognitive Prosthetic — Adaptive Interface by Neurology

*April 18, 2026*

## Core Idea

Smart glasses as a cognitive prosthetic — not just showing information about your environment but adapting HOW it presents information based on how your brain works. Same data engine underneath, different presentation layer per neurotype.

## The Insight

The same prediction engine (pantry, consumption rates, reactions, shopping list) powers all modes. The difference is what to show, when, and how much. The interface adapts to the person, not the other way around.

## Neurotype Modes

### ADHD
- Time awareness: "You've been in this aisle for 4 minutes. 3 items left."
- Visual breadcrumbs showing your route so you don't backtrack (working memory support)
- Dimming/hiding non-list items to reduce overwhelm (stimulus filtering)
- Gentle impulse check: "You picked up chips. That's not on your list."
- Task chunking: "Next: eggs. Aisle 4, turn left."
- Dopamine hits on completion of each item

### Anxiety / OCD
- Reassurance without rumination: "You already checked this. It's safe."
- Limiting re-scans of the same product (interrupt compulsive checking)
- Calming overlay when heart rate spikes (paired with wearable)
- Reaction history as evidence: "You've eaten this 12 times with no issues"

### Autism / Sensory Processing
- Noise level indicator, crowd density warnings
- Quiet-aisle routing: "This aisle is empty right now"
- Minimal visual clutter in the overlay itself
- Predictable, consistent UI — no surprises

### Depression / Low Executive Function
- Absolute minimum viable trip: "You need 4 things. Here's the fastest path."
- No overwhelming data — just gentle next-step guidance
- Celebrating completion: "Done. 12 minutes. Faster than last time."
- Low-energy mode: fewer interactions, more automation

## Bigger Than Grocery

Grocery is just the first context. The same adaptive framework applies to:
- **Cooking** — step-by-step with ADHD chunking, or minimal-overwhelm mode
- **Cleaning** — task breakdown, time estimates, gentle redirection
- **Work tasks** — focus mode, break reminders, priority surfacing
- **Social situations** — conversation cues, exit strategies, sensory monitoring
- **Medical appointments** — question lists, recording, follow-up reminders

Anywhere executive function is the bottleneck, an adaptive AI interface layer that knows your neurology could help.

## Market Position

Nobody is building adaptive-by-neurotype AR interfaces. Accessibility features exist (font size, contrast, screen readers) but that's physical disability accommodation, not cognitive adaptation. The mental health / neurodivergent population is massive, underserved, and increasingly self-aware about what they need.

## What Needs to Exist First

1. The prediction engine (grocy-predict → exocortex pantry)
2. The health/reaction data layer (already building)
3. AR glasses hardware (arriving 2025-2026)
4. User profiles with neurotype preferences
5. Research partnership — validate which adaptations actually help (this is where the PhD matters)
